# SNSY-Bot
## The custom moderation bot for your server.
---
## Features
### Slash Commands
- Soft-Ban/Mute/Kick
- Soft-Unban/Unmute
- Purge
- User info / Server info
- Status / help
- User history

### Other
- User/Staff permissions
- Ticket unban
- Logs - warns / notifications - unban tickets
- Setup Commands
